# Archive Index

This folder holds historical material that no longer represents the active NEXUS baseline.

It exists so the working repo stays small and truthful while preserving prior context when needed.

## What Lives Here

- `docs/hackathon/`: submission-era docs that described the earlier public demo phase
- `docs/historical-plans/`: superseded phase plans, execution maps, and earlier roadmap documents
- `docs/superpowers-specs/`: historical design specs generated for completed phases
- `artifacts/demo-video/`: older recorded demo assets for the `v2` submission period
- `artifacts/nexus-v2-presenter-pack/`: older presentation build assets

## Rule

If an active doc in `/docs` or the running product conflicts with something in this folder, trust the active doc or the current code.

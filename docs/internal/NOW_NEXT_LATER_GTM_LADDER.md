# NEXUS Now / Next / Later GTM Ladder

Current as of 2026-06-15.

## Now

Sell NEXUS as:

- support-to-engineering investigation workflow
- recurring customer-facing outages
- five bounded outage families
- runtime-aware, debugging-guided, human-governed triage

## Next

Expand only if it strengthens the same wedge:

- more repeatable tenant onboarding
- stronger fresh-incident quality
- stronger engineering trust and proof surfaces

## Later

Do not claim until implemented:

- arbitrary environment reproduction
- arbitrary repository debugging
- universal multi-stack incident response
